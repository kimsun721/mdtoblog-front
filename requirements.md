# Requirements — 백엔드 미구현 기능

디자인 HTML(`mdtoblog(1).html`)에 존재하지만 현재 백엔드 API에 없어 하드코딩으로 처리된 기능 목록입니다.

---

## ✅ 구현 완료

### 플랫폼 통계 (히어로 섹션)
`GET /api/stats` 구현됨. 홈 히어로 배너에서 실제 API 연동.

### 포스트 정렬
`GET /api/post?sort=views|created_at|likes` 구현됨. 인기 글 탭 및 사이드바 TOP 5에서 사용.

### 활발한 개발자 (사이드바)
`GET /users/top?limit=4` 구현됨. 홈 사이드바에서 실제 API 연동.

### 댓글 좋아요
`POST/DELETE /comment/:id/likes` 구현됨.

---

## ❌ 미구현 (하드코딩 유지)

### 1. 인기 키워드 (사이드바)

**위치**: 홈 사이드바 "🔍 인기 키워드"

현재 상태: 키워드 목록 하드코딩 (`['Rust', 'TypeScript', 'NestJS', ...]`)

클릭 시 해당 키워드로 검색은 동작함.

---

### 2. 팔로워 수 (프로필 페이지)

**위치**: 프로필 페이지 통계 섹션

현재 상태: 프로필 페이지에서 제거됨 (구현 안 함)

---

### 3. 포스트 뱃지 / 태그

**위치**: 포스트 카드 및 상세 페이지

현재 상태: `post.repoName`을 뱃지로 대체 사용 중. 백엔드 Post 타입에 `tags` 필드 없음.
